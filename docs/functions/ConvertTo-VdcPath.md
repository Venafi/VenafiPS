# ConvertTo-VdcPath

## SYNOPSIS
Convert GUID to Path

## SYNTAX

```
ConvertTo-VdcPath [-Guid] <Guid> [-IncludeType] [[-VenafiSession] <PSObject>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Convert GUID to Path

## EXAMPLES

### EXAMPLE 1
```
ConvertTo-VdcPath -Guid [guid]'xyxyxyxy-xyxy-xyxy-xyxy-xyxyxyxyxyxy'
```
