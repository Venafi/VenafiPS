# Get-TppVersion

## SYNOPSIS
Get the TPP version

## SYNTAX

```
Get-TppVersion [[-VenafiSession] <VenafiSession>] [<CommonParameters>]
```

## DESCRIPTION
Returns the TPP version

## EXAMPLES

### EXAMPLE 1
```
Get-TppVersion
```

Get the version
